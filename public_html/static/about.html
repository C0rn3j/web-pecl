About this site

This site has been created and is maintained by a number of people, which are listed on the credits page. If you would like to contact them, you can write to pear-webmaster@php.net.

It has been built with Apache, PHP, MySQL and some (as you might have guessed) PEAR packages. Additionally we have started to use a set of utility classes. We call it Damblan. The source code of the website is available via CVS.
