In the press

International edition of PHP-Magazine

We welcome the new international edition of PHP Magazine. In this high quality first issue you can find a very good introduction to PEAR::MDB written by Lukas Smith. Also included is an usefull introduction on how to write PHP extensions by Zeev Suraski.

PHP-Magazin

The 1 / 2003 issue of PHP-Magazin has many well written articles about PEAR. Alexander Merz wrote a nice PEAR resume. He also wrote an usefull introduction of setup, administration and usage of PEAR. Hartmut Holzgraefe shows us how to use the PEAR::HTTP_WebDAV_Server. Richard Samar introduces XML databases XPath and XQuery. He also writes about using PEAR::XDB modules, especially the Tamino driver.

We are pleased to see many articles about our project If any of you read articles or news about PEAR, feel free to send us the references and we'll add them to the weekly news.

PEAR-DEV traffic

DB_DOM ready for CVS

Sandro Zic notified their allready pear-dev approved class for DOM-like management of XML content in a defined RDBMS table structure has been pearified and is ready to be committed into CVS. Unfortunately they haven't decided the final name yet and are waiting for feedback.

Some proposed names were: DB_DOM, DB_XML, DB_XML_DOM, XML_DB and XML_DB_DOM.

PEAR Foundation?

There was some discussion on whether there should be be a foundation for sponsoring PEAR development or not. There are similar foundations sponsoring big projects such as FreeBSD and Perl. However, PEAR is still in the small league of open source projects and a foundation would be a bit of an overkill. The current per-developer wishlist style approach should be adequate.

New and Proposed Packages

Net_LMTP

Damian Alejandro Fernandez Sosa suggested a class which is cabable of delivering email via LMTP (Local Mail Transfer Protocol, defined in RFC 2033). The proposed name was Net_LMTP which serves the purpose perfectly. Net_LMTP is based upon Net_SMTP.

PhpDocumentor

PhpDocumentor, which has been downloadable from http://www.phpdoc.org/ for a while is now also available via PEAR.

Gtk_VarDump

Gtk_VarDump made also it first PEAR release.
