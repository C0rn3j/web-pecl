Novos Lançamentos

Estável

  • XML_Transformer - transforma tags XML em código PHP ou texto (conceito similar ao XSLT)
  • XML_RPC - uma API para gerenciar Remote Procedure Calls
  • PEAR - o pacote base do PEAR

Beta

  • Translation - uma API para gerenciar I18N de recursos em string
  • Config - uma API para processar e editar configurações de fontes de dados

Alpha

  • LiveUser - autenticação de usuário e gerenciamento de permissões
  • XML_XSLT_Wrapper - camada de abstração para a transformação XSL

Tráfego na Lista PEAR-DEV

Categoria Perm?

Markus Wolff solicitou uma nova categoria para o LiveUser chamada Perm. Não houve concenso e Markus lançou o LiveUser sob a categoria Authentication.

Vai PEAR, vai!

Stig Bakken registrou os domínios go-pear.org|.com|.net. Eles servirão para o script go-pear, o qual instala a base atual do sistema PEAR incluindo o instalador. Se você desejar um PEAR funcionando no seu sistema basta rodar no seu prompt:

lynx -source go-pear.org|php

Privacidade do PEAR

Derick Rethans notou alguns possíveis problemas na página web do PEAR. Se você baixar um pacote, seu IP será guardado para fins estatísticos. Isto poderia ferir leis em alguns países europeus. Não se decidiu uma forma para lidar com isto - nós somos programadores, não advogados. Se você sabe mais do que nós sobre este assunto escreva uma nota para a lista de discussão pear-dev.
